# North Star

## Mission

Crafter enables developers to ship changes with AI assistance while preserving human control at every important decision point.

## Target Outcomes

- predictable, reviewable change workflow (plan → implement → verify → review)
- low-friction project context management via `.crafter/` files
- safer AI contribution through explicit approval gates
- minimal operational overhead for solo developers and small teams

## Product Principles

1. Human-in-the-loop by default
2. Surgical changes over broad refactors
3. Explicit state and traceability over implicit behavior
4. Skills-first workflow definitions with reusable conventions
5. Keep the core lightweight and easy to adopt

## Success Signals

- contributors can start and complete a change without guessing process expectations
- context files and docs remain aligned with actual repository behavior
- workflow quality improves over time through consistent review and verification steps
